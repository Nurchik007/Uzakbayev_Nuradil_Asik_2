Object-Oriented Programming — Assignment II
This is a student project for the Object-Oriented Programming (OOP) course.
The goal is to practice and demonstrate the principles of OOP using a Java application.

Project Structure
.idea/ — configuration files for IntelliJ IDEA

src/ — source code written in Java

.gitignore — list of files excluded from version control

untitled.iml — IntelliJ IDEA project configuration file

Implemented OOP Concepts
Classes and Objects — modeling real-world entities

Encapsulation — hiding internal implementation with access modifiers

Inheritance — building class hierarchies

Polymorphism — method overriding and interface usage

Abstraction — defining common behaviors and characteristics

Installation and Run
Clone the repository:
git clone https://github.com/Nurchik007/Uzakbayev_Nuradil_Asik_2.git
Open the project in IntelliJ IDEA

Make sure you have Java 17 or higher installed

Run the application via src/Main.java
